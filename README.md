🎯 What is it?
Network Analyzer Pro is a browser-based tool that automatically parses network configs, builds interactive topology maps, validates configurations, simulates paths, and exports results - all with stunning animations!

✨ Features
Feature	Description
📁 Upload	Drag & drop .dump, .txt, .cfg files
🎨 Auto-Topology	Automatically detects network connections
🌐 Interactive Map	Drag, zoom, explore network graph
✅ Validation	Detects duplicate IPs, MTU mismatches, errors
🔄 Simulation	Find paths and test link failures
🔍 Search	Find and highlight devices
📸 Export	PNG, JPG, JSON formats
🏷️ VLANs	Detect and display VLAN configs
🔍 Compare	Side-by-side config comparison
📊 Stats	Live device, link, error counts
🚀 Quick Start
Open index.html in your browser

Click "Load Error Sample" or "Load Clean Sample"

Click "Analyze Network"

Explore results in tabs!

Keyboard Shortcut
Ctrl + Enter - Quick analyze
